Equation 4: Number Density using Number Fraction

Equation:

\[N_i = \frac{\gamma_i \rho_x N_A}{A_x}\]

Variables:

  • \(N_i\): Number density of isotope i [particles/cm^3]
  • \(\gamma_i\): Abundance (or number fraction) of isotope i
  • \(\rho_x\): Elemental mass density [g/cm^3]
  • \(N_A\): Avogadro's number
  • \(A_x\): Atomic mass of element x

Description:

Number density (particles per unit volume) for isotope i calcuated using number fraction.
